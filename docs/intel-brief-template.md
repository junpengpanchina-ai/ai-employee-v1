# /intel 输出模板（趋势分析师 + AI 总经理）

> **定位**：不是「新闻摘要器」，而是 **趋势分析师 + AI 总经理 + 老板情报入口**。  
> **要解决**：不是「发生了什么」，而是 **今天什么最值得看、为什么、和你有什么关系、下一步怎么动**。  
> **源码常量**：`apps/orchestrator-service/src/intelPrompts.js`（`INTEL_SYSTEM_PROMPT`、`buildIntelUserPrompt`）

---

## 一、输出目标（`/intel` 必须做到的 5 件事）

1. **抓大势**：不罗列碎资讯  
2. **抓主矛盾**：当前最关键冲突点  
3. **看结构流向**：资源、流量、供需往哪动  
4. **看竞争位置**：谁在上位 / 失位  
5. **给动作建议**：今天要不要跟、跟哪条  

---

## 二、老板可见格式（三档）

### 标准版（默认，六段）

与代码中 **`INTEL_SYSTEM_PROMPT`** 结构一致：

```text
今日情报简报

一、今日最重要的变化
[用 1 句话说清今天最值得关注的趋势]

二、主要矛盾
[当前最核心的冲突/变化点是什么]

三、结构流向
[资源、流量、供给、需求、政策、平台重心在往哪里移]

四、竞争位置
[谁在占位，谁在失位，谁值得继续盯]

五、与你的关系
[这件事和你当前业务/系统/机会判断有什么关系]

六、今日动作
1. [最优先动作]
2. [可选动作]

如需展开，我可继续拆：
- 行业影响
- 平台机会
- 竞品动向
- 可执行动作
```

### 极简版

```text
今日情报简报

1. 重点变化：[一句话]
2. 核心判断：[一句话]
3. 对你影响：[一句话]
4. 今日动作：[一句话]
```

### 强行动版

```text
今日情报简报

结论：
[先说最重要结论]

判断：
[为什么这是今天最该盯的事]

动作：
1. [现在就做]
2. [可以观察]
3. [暂不投入]
```

### 三档策略（产品可选）

| 档位 | 适用 |
|------|------|
| 默认速读 | Telegram 直接看 |
| 经营判断版 | 要进一步判断「值不值得投」 |
| 极短一句话 | 自动推送 / 提醒 |

## 三、System Prompt

**以仓库为准**：`apps/orchestrator-service/src/intelPrompts.js` 中的 **`INTEL_SYSTEM_PROMPT`**（含资治通鉴 / 毛选 / 社会经济学 / 商业 / 投资五维，及六段输出约束）。

---

## 四、User Prompt 调用方式

使用 **`buildIntelUserPrompt(intelItemsText)`**，把上游拉取并整理后的**情报正文**拼入。

背景默认包含（已在函数内固定）：

- Telegram 作为老板入口  
- 关注 AI 自动化、平台变化、可产品化机会等  
- 要「今天最该看什么、该不该跟、先做什么」  

---

## 五、上游情报输入格式（建议）

### JSON（推荐）

```json
[
  {
    "title": "标题1",
    "source": "来源1",
    "published_at": "2026-04-12T09:00:00Z",
    "summary": "这条资讯的原始摘要",
    "url": "https://..."
  }
]
```

### 纯文本（最低限度）

```text
[1]
标题：...
来源：...
时间：...
摘要：...

[2]
标题：...
来源：...
时间：...
摘要：...
```

## 六、五镜头压缩（内部自检，不出现在输出里）

每次摘要可过这五镜（逻辑等价于 system prompt 中的五维）：

| 镜头 | 自问 | 输出句式示例 |
|------|------|----------------|
| 趋势与时机 | 上升/扩散/拐点/尾声？ | 「这件事处于……阶段。」 |
| 主要矛盾 | 只能盯一件事时盯谁？ | 「当前主要矛盾不是……，而是……。」 |
| 结构与资源流向 | 资源、平台重心往哪动？ | 「资源正在从……流向……。」 |
| 竞争与位置 | 谁受益/受损/卡位？ | 「……在占位，……在失位。」 |
| 资金与注意力 | 长期趋势还是噪音？ | 「注意力正集中到……。」 |

---

## 七、输出禁区（建议写死）

- `<redacted_thinking>`、think 块  
- 「我开始分析……」「我正在判断……」「以下是我的思考过程……」  
- 逐条复述原始资讯、新闻播报腔、长篇研究报告腔、空话套话、长篇英文分析  

（编排侧可用 `replyPolicy.sanitizeReplyText` 再挡一层。）

---

## 八、合格校验（一句话）

> **这段话有没有让我更快知道「今天最该看什么、为什么、该做什么」。**  
> 有 → 合格；没有 → 仍是资讯搬运。

## 九、与 WorldMonitor 接入的关系

1. 先 **WM 第三服务在线** + 确定 **取料契约**（API / RSS / 导出 / 临时抓页）。  
2. **`fetchWorldMonitorFeed`** 把材料整理成 **§五** 格式。  
3. **`summarizeIntel`** = `buildIntelUserPrompt(...)` + `INTEL_SYSTEM_PROMPT` + GRSAI。  
4. 再接到 Telegram **`/intel`**。  

执行勾选见 [`worldmonitor-execution-checklist.md`](./worldmonitor-execution-checklist.md)。
